This is the Readme file:
Make a new env for the code to run locally, and then to run locally you need two terminals, one for the backend and one for the frontend.
to run backend Terminal : python -m uvicorn main:app --host 0.0.0.0 --port 8000 --reload
To run frontend terminal : python -m http.server 3000

and then go to localhost:3000 on your system the website will be running
